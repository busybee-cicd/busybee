Class BusybeeUserConfig

Busybee Configuration. Mapped to Typescript from user-provided config.js/json. The top-level of configuration, the fields in the BusybeeUserConfig are responsible for defining the resources available for environment provisioning, TestSuites that will be run with-in environments and registering reporters for result analysis

{
  envResources: EnvResourceConfig[],
  onComplete: 'onComplete.js',
  testSuites: TestSuiteConfig[],
  reporters: []
}
